# 项目测试方式：

1. 安装依赖 `npm install` 或者 `pnpm install`
2. 执行 `npm run dev`
3. 浏览器中打开本地服务器地址: http://localhost:5173/
4. 输入 openrouter 后台给的 api key，点击 `start chatting`
5. 页面自动跳转到聊天页面，即可测试其他功能
